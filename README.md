# Silver-Screen 🎥

## Ultimate AI Movie Maker for NFT Cinematic Universes

**Goal**: Build the best free/open AI-powered long-form movie creation app - better than Pixel Bunny, rivaling or exceeding Grok Imagine quality for scenes, enabling your 80-minute NFT movie.

### Vision
- **Multi-Model Orchestrator**: Seamlessly use dozens of top AI models (Flux, SD3, Kling-like via APIs, Luma, Runway, open-source via HF/Replicate/ComfyUI/Fal).
- **Long-Form Consistency**: Character sheets, IP-Adapter, ControlNet, story memory, scene graph.
- **Full Pipeline**: Script → Storyboard → Scenes (img2vid) → Edit (timeline) → Sound (ElevenLabs/Suno) → Render → NFT assets/mint.
- **UI like pro editor + AI copilot**.

**Tech**: Next.js 15 + TypeScript + Tailwind + shadcn/ui + Replicate/HF APIs + Zustand + React Flow for graph + Remotion/FFmpeg.

## Quick Start
1. `git clone ...`
2. `npm install`
3. Add API keys in `.env.local` (Replicate, Groq, HF, etc. - many free tiers)
4. `npm run dev`
